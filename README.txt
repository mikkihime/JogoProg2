O jogo necessita de todos os arquivos desta pasta para funcionar corretamente!
Ele foi programado usando Python 2.7 e a biblioteca Pygame, logo necessita desses dois itens para
funcionar.
Para jogar, basta clicar duas vezes em "jogo.py"
O jogo possui 4 fases, cada uma com um boss próprio, e um boss final. 
A cada boss derrotado, você ganha um power up para a sua nave.
Bom divertimento!
